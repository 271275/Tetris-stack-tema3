Este repositório contém a entrega do Desafio Tetris Stack – Tema 3.

 Objetivo do Tema 3
O desafio Tetris Stack tem como propósito demonstrar que o aluno compreender os conceitos iniciais de versionamento, organização e controle de arquivos utilizando GitHub como ferramenta acadêmica e profissional.

A atividade consiste em criar um repositório público e registrar neste arquivo (README.md) as informações básicas da entrega.

Este repositório demonstra a criação de repositório público no GitHub, uso do arquivo README.md, registro e documentação da atividade e ompreensão do conceito de versionamento e armazenamento na nuvem
